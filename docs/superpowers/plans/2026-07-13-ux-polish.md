**Goal:** Fix leftover spacing/consistency issues on the listing detail page, homepage, and footer, and add two new homepage sections ("Jak to działa" and a seller CTA banner).

**Architecture:** Pure frontend changes to three existing React components (`ListingDetailPage.tsx`, `HomePage.tsx`, `Footer.tsx`) plus the two small prop-wiring points in `App.tsx` that connect `Footer` and the new `HomePage` CTA banner to the app's existing navigation/auth-modal handlers. No test framework is configured in this project (`package.json` has no test script) — verification throughout is `npm run typecheck`, `npm run build`, and manual browser checks via the dev server, matching how every other feature in this codebase has been verified.
